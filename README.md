# 🐦 Pigeon Chat Server

A very simple, minimal chat server & client written in **Go**.

---

## 🚀 run
- **Start the server:**
    ```bash
    pigeon -l <port>
    ```

- **Connect:**
    ```bash
    pigeon -c <host>:<port>
    ```

### 🛠️ build from the source

    ```bash
    git clone https://github.com/falajh/pigeon
    cd pigeon
    go build
    ```
